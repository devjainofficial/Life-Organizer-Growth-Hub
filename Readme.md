## 📖 About The Project

**Life Organizer & Growth Hub** is a beautifully designed, feature-rich web application that helps you organize your daily tasks, track your learning progress, maintain habits, and stay focused. Built with vanilla HTML, CSS, and JavaScript, it offers a clean, intuitive interface with multiple theme options to suit your preference.

Whether you're a student, professional, or anyone looking to boost productivity, this tool provides everything you need in one calming, distraction-free space.

### ✨ Key Highlights

- 🎨 **Beautiful UI** - Modern, clean design with smooth animations
- 🌓 **Multiple Themes** - Default, Calming Blue, and Sunset Orange themes
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
- 💾 **Local Storage** - Your data is saved automatically in your browser
- ⚡ **Fast & Lightweight** - No dependencies, pure vanilla JavaScript
- 🔒 **Privacy First** - All data stays on your device

## 🏁 Getting Started

### Prerequisites

All you need is a modern web browser! No installations or dependencies required.

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/whatsupsumit/Life-Organizer-Growth-Hub.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Life-Organizer-Growth-Hub
   ```

3. **Open the application**
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

4. **Access the app**
   - Open your browser and visit `http://localhost:8000`
   - Start organizing your life! 🎉

## 💻 Usage

### Adding Tasks
1. Type your task in the "Add a new task" input field
2. Click "Add Task" or press Enter
3. Click on the task to mark it as complete
4. Use the edit (✏️) button to modify tasks
5. Use the delete (❌) button to remove tasks
6. Click "🗑️ Remove Completed" to clear all completed tasks

### Tracking Skills
1. Add a skill name in the Learning Tracker
2. Use the + and - buttons to adjust skill levels
3. Watch your progress bar grow to 100%

### Using the Focus Timer
1. Click "Start" to begin a 25-minute focus session
2. Use "Reset" to restart the timer
3. Toggle sound notifications as needed

### Switching Themes
- Click the theme toggle button in the top-right corner
- Cycle through: Default → Calming Blue → Sunset Orange
